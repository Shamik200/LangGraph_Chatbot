## � **Quick Start Guide**

### 📋 **Prerequisites**
- Python 3.8+ (We recommend 3.12 for optimal performance)
- Google AI API Key ([Get yours FREE](https://makersuite.google.com/app/apikey))
- Git (obviously)

### ⚡ **Lightning Setup** (60 seconds!)

```bash
# 1. Clone the beast
git clone https://github.com/Shamik200/LangGraph_Chatbot.git
cd LangGraph_Chatbot

# 2. Create virtual environment
python -m venv myenv

# 3. Activate environment
# Windows:
.\myenv\Scripts\Activate.ps1
# Linux/Mac:
source myenv/bin/activate

# 4. Install dependencies
pip install -r requirements.txt

# 5. Setup environment
cp .env.example .env
# Edit .env and add your GOOGLE_API_KEY

# 6. Launch! 🚀
streamlit run streamlit_frontend.py
```

### 🌐 **Access Your AI Assistant**
- Open your browser to `http://localhost:8501`
- Start chatting with your new AI companion!
- Try asking it to search the web, solve math problems, or just have a conversation!
